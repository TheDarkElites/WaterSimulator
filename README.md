# Setup
## Ubuntu / Debian
Figure out how to get CMake Version >= 4.1. The `apt` version is too outdated. I got mine bundled with CLion.
Install OpenGL and supporting packages
```bash
sudo apt install libgl1-mesa-dev libglu1-mesa-dev freeglut3-dev libglew-dev
```
